aplikacja stworzona do zarzadzania kotami (domyslnie kotami, mozna rowniez dodawac inne zwierzeta w ramach wyjatku hehe). umozliwia planowanie planowanie zycie kota i opiekonow od najprostszych czynnosci takich jak zabawa z kotem az po aktywnosci wymagajace wiekszego planingu: wizyty u weterynarza czy podawanie lekow (dodatkowo uniemozliwiamy utuczenia kota poprzez podawanie mu sniadania przez 3 rozne osoby w ciagu dnia)


/login ustawia sesję i rolę.
/cats pobiera listę kotów dla użytkownika.
/details pobiera kota + jego aktywności + opiekunów.
/schedule pobiera aktywności dla widocznych kotów.
/logs pobiera logi dla widocznych kotów.
/settings (admin) zarządza userami.
/caregivers zapisuje powiązania cat ↔ caregiver.
/reports liczy podsumowania z logów/aktywności.
/profile obsługuje dane konta.
